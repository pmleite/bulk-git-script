# bulk-git-script

This script does GIT bulk operation like status, pull, push


# Examples

Bulk git status operation 

```
bulk-git --bulk --status main 
```

Bulk git pull operation 
 
```
bulk-git --bulk --pull main 
```

Bulk git push operation 
 
```
bulk-git --bulk --push main 
```





# Compiled with shc

```
shc -f bulk-git.sh
```
